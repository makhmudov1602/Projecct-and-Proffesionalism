# Nishon Arena Frontend

Bu versiya imtihon tizimi emas.

Asosiy yo'nalish:
- archery va rifle competition
- ishtirokchilarni qo'shish
- session yaratish
- live leaderboard
- kamera preview va score capture
- PDF export

## Ishga tushirish
```bash
npm install
npm run dev
```

## Asosiy o'zgarishlar
- Face ID oqimi olib tashlandi, o'rniga player registration qo'yildi
- exam / talaba / baholash terminlari arena formatiga almashtirildi
- archery va rifle mode qo'shildi
- scoreboard logikasi soddalashtirildi
- backend ulanmasa ham demo/offline flow ishlaydi
