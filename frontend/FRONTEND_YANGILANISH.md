Frontend yangilanishlari:

- login sahifasi to'liq qayta dizayn qilindi
- bosh sahifa akademik dashboard ko'rinishiga o'tkazildi
- eski logolar va fon rasmlari olib tashlandi
- talabani ro'yxatdan o'tkazish bloki soddalashtirildi
- imtihon boshqaruv paneli aniqroq ko'rinishga keltirildi
- talabalar kartasi qayta ishlanib, kamera, progress va natija bloklari yaxshilandi
- restart logikasi kamera baseline tozalash bilan yaxshilandi
- natijalar jadvali summary kartalar va yaxshiroq PDF eksport bilan yangilandi

Ishga tushirish:
1. papkani oching
2. npm install
3. npm run dev
